# 📍 Indoor Positioning System (Android)

## 🚀 Overview

This Android application collects Wi-Fi signal data to build an indoor positioning system.
It helps map indoor spaces and prepare data for machine learning models.

---

## ✨ Features

* 📡 Scan nearby Wi-Fi networks
* 🏷️ Label data using room names (Room1, Lab, etc.)
* 🗺️ Load map images from gallery
* 📍 Select points on map
* 📊 Save data as CSV file
* 📤 Export CSV for ML training
* 🧭 Open Google Maps for location reference

---

## 📱 App UI

* Clean "Area Mapper" interface
* Map preview with touch interaction
* Grid overlay for positioning
* Easy-to-use controls

---

## 🛠️ Tech Stack

* Kotlin (Android)
* Android SDK
* Wi-Fi Manager API
* File Storage (CSV)
* Custom View (Grid Overlay)
